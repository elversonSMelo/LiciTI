# Projeto - LiciTI
# Universidade Federal de Pernambuco - Centro de Informática
# IF977 - Engenharia de Software

	
# DESCRIÇÃO:

    O foco do projeto é desenvolver um SaaS responsável por gerenciar empresas terceirizadas que possam ou não
	participar de licitações promovidas por órgãos do governo, verificando se estão impugnadas ou não, de acordo
	com base de dados provida pelo Tribunal de Contas da União. Nosso público alvo são as empresas e órgãos públicos
	que buscam a prestação de serviços e necessitam de agilidade na verificação das propostas oferecidas.
